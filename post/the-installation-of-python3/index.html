编译安装 Python3 以及 sqlite3 模块

2018-06-12
编程

Contents

    • 编译安装 Python3
    • 安装 sqlite3 模块

编译安装的好处

  • 有些系统应用依赖自带的 Python 环境，若编译安装到独立目录，可起到隔离作用；
  • 采用 yum、apt 等包管理工具安装，不利于管理，且安装位置无法自控；
  • 线上生产环境，采用编译安装，减少依赖性。

编译安装 Python3

3

cd /usr/local/src
wget https://www.python.org/ftp/python/3.6.8/Python-3.6.8.tgz
tar xvf Python-3.6.8.tgz
cd Python-3.6.8
./configure --prefix=/usr/local/python368
make && make install

采用 root 用户，安装好的 Python 环境位于 /usr/local/python368 目录下面。

安装 sqlite3 模块

使用 Sqlite3 数据库时，如果出现 sqlite3 module not found 异常, 需要先安装 sqlite-devel

yum install sqlite-devel

然后，重新编译安装 Python3

./configure --prefix=/usr/local/python368 --enable-loadable-sqlite-extensions
